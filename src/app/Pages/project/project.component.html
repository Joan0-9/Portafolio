Nombre del Proyecto: SICOD

Es un aplicativo web que fue desarrollado en conjunto con dos desarrolladores más, con el fin de facilitar la gestión de documentación presentada por los docentes en la seccional Palmira, Valle del Cauca, de la Universidad del Valle para el proceso de contratación de cada periodo académico.

Etapa de contratación cliente

Etapa legal con un contrato de por medio donde se realizaron reuniones con el cliente para entender su problema y poder proponer una solución.

Las tecnologías implementadas para esta etapa fueron:

  • Figma

    • Herramienta de diseño utilizada para la creación visual de la aplicación.

  • StarUML

    • Herramienta de diseño utilizada para la creación de diagramas útiles para la creación de la solución del proyecto.

  • Google Drive

    • Herramienta de almacenamiento en la nube ofrecida por los servicios de Google utilizada para el alojamiento de los documentos y archivos.

  • Google Meet

    • Herramienta de videollamadas ofrecida por los servicios de Google que fue utilizada para reuniones con el cliente y equipo de trabajo.


Etapa de desarrollo

Etapa enfocada en el desarrollo de código teniendo en cuenta lo acordado con el cliente

Las tecnologías implementadas para la creación del proyecto fueron:

  • Angular

    • Framework destinado al desarrollo de la parte visual (FrontEnd).

  • Spring Boot

    • Framework destinado al desarrollo de la parte que va del lado del servidor (BackEnd).

  • MySQL

    • Sistema de gestión de bases de datos utilizado para el manejo de estos.

  • GitHub

    • Plataforma en la cual está almacenado el software.

  • Postman

    • Herramienta de apoyo para la creación, prueba de APIs.
